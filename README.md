# Lab 4: Carga de modelo

Laboratorio de renderizado 3D de una nave espacial.

## Nave
https://github.com/JPLopez23/Lab4G/issues/1

## Controles

| Tecla | Acción |
|-------|--------|
| **Flechas** | Rotar cámara |
| **Q / E** | Girar nave sobre sí misma |
| **W / S** | Zoom in / out |
| **R** | Resetear cámara y rotación |
| **1-5** | Cambiar dirección de luz |
| **ESC** | Salir |
