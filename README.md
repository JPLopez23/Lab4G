## Controles

| Tecla | Acción |
|-------|--------|
| **Flechas** | Rotar cámara |
| **Q / E** | Girar nave sobre sí misma |
| **W / S** | Zoom in / out |
| **R** | Resetear cámara y rotación |
| **1-5** | Cambiar dirección de luz |
| **ESC** | Salir |
